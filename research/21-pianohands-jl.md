## What It Is

A Julia package for **predicting hand assignments in piano MIDI files** using neural networks. Given a MIDI file, it determines which notes should be played by the left hand vs right hand.

**This is not a gesture recognition project.** It operates on MIDI data, not webcam input.

## Relevance

Low relevance to the core goal (webcam gesture → MIDI). The JuliaMusic organization has 73 followers which would normally be a positive signal, but the project itself doesn't fit the use case.

Potentially interesting if we later build MIDI post-processing features (e.g., analyzing or splitting captured MIDI output), but that's far out of scope.
