## What It Is

A highly focused single-purpose tool: measures thumb-to-index-finger distance and converts it to a MIDI CC value.

**Stack:** JavaScript (likely MediaPipe JS + Web MIDI API)

## Relevance

Low signal — 0 stars, 0 followers. But the specificity is notable: pinch distance → CC is the most atomic gesture-MIDI primitive, and this project does only that. Useful as a minimal reference implementation if we need to isolate just that one parameter.

Not worth prioritizing over more complete projects.
