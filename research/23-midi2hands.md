## What It Is

A Python app that uses ML to **separate left and right hand notes in MIDI files**. Given an existing MIDI file, it classifies each note as belonging to the left or right hand. Users can train custom models on their own MIDI files.

**This is not a gesture recognition project.** Like PianoHands.jl, it operates on MIDI data rather than webcam input.

## Relevance

Low relevance to the core goal. The name is misleading — this is MIDI → hand assignment, not hand gestures → MIDI.

Not worth further investigation for the gesture input use case.
