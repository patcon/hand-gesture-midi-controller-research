# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a research and prototyping project for gesture recognition via webcam (primarily MediaPipe) mapped to a programmable virtual MIDI device. The intended output is an app using native APIs (Electron, React Native, or similar). Rust is welcome despite no prior experience.

## Research Priorities

When researching or recommending approaches, favor:
- MediaPipe-based gesture recognition approaches
- Python, TypeScript/JS, or Rust implementations
- Newer projects with high star counts
- Building on existing libraries rather than from scratch
- Work from GitHub users with high follower counts or other highly-starred packages (social trust signal)

The project should be usable by others — ideally as both a reusable library and standalone utility.
